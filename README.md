# Online Grocery Delivery Platform

## Summary
A full-stack grocery delivery application built with React and Node.js/Express. The platform allows users to browse products, manage carts and checkout, handle subscriptions and loyalty points, and includes an admin dashboard for product and order management.

## Key Features
- Product browsing with categories and images  
- Cart and checkout flow  
- User and admin authentication  
- Address and profile management  
- Subscriptions and loyalty points system  
- Admin panel for managing products and orders  
- Responsive UI for desktop and mobile  

## Tech Stack
- **Frontend:** React, JSX, Context API, CSS  
- **Backend:** Node.js, Express.js  
- **Data:** JSON-based storage (`db.json`) for development  

## Project Structure
- **Frontend:** components, admin, loyalty, assets, styles  
- **Backend:** controllers, models, routes, middleware, config
